Today I Learned…

  • 대학미적분. 미분계수.
    • 미분가능하다 == 미분계수가 존재한다 == $f(x)$를 수직이 아닌 직선으로 근사할 수 있다.
      • 근사한다는 말뜻이 잘 이해가 안된다. 승혀니햄이 공유해주신 영상을 차차 공부해보자.

  • 나의 오랜 아킬레스건같았던 삼각함수를 드디어 공부했다…!
    • 관련하여 정리한 내용은 이 포스트 를 참고하시라.

  • 분명 CLI환경에서는 도커가 돌아가고 있는데, Docker Desktop은 stop인 상태로 떠서 찾아보다가 알게 된 놀라운 사실…
    • 리눅스 환경에서 Docker-desktop과 Docker-engine은 다르다!!!!
      • 처음 깨닫게 해준 글 , 도커 공식 문서
    • 간단히 말해서 Docker-desktop은 가상머신 위에서 돌아가고, docker-engine은 그냥 host OS에서 바로 돌아가는 것 같다!
      • 아마 플랫폼간의 호환성을 고려한다면 docker desktop을 쓰는 게 좋을 것 같은데 어떻게 해야 Docker-desktop을 쓸 수 있는건지는 아직 미궁이다…
