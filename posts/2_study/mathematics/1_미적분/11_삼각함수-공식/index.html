[미적분] 삼각함수 공식들

 2023-04-30  562 words   2 minutes

  • 2. 삼각함수의 덧셈정리

1. 삼각함수의 성질

  • x축 대칭 (음각 공식)
    • $\cos({-\theta}) = \cos{\theta}$
      • 따라서 코사인함수는 우함수.
    • $\sin({-\theta}) = -\sin{\theta}$
      • 따라서 사인함수는 기함수
    • $\tan({-\theta}) = -\tan{\theta}$
      • 따라서 탄젠트함수는 기함수.

  • y축 대칭 (보각 공식)
    • $\cos({\pi - \theta}) = -\cos{\theta}$
    • $\sin({\pi - \theta}) = \sin{\theta}$
    • $\tan({\pi - \theta}) = -\tan{\theta}$

  • 원점 대칭
    • $\cos({\pi + \theta}) = -\cos{\theta}$
    • $\sin({\pi + \theta}) = -\sin{\theta}$
    • $\tan({\pi + \theta}) = \tan{\theta}$

  • $y=x$축 대칭 (여각 공식)
    • $\cos({\frac{1}{2}\pi - \theta}) = \sin{\theta}$
    • $\sin({\frac{1}{2}\pi - \theta}) = \cos{\theta}$
    • $\tan({\frac{1}{2}\pi - \theta}) = \frac{1}{\tan{\theta}} = \cot{\theta}$


2. 삼각함수의 덧셈정리

  • $\cos({\alpha + \beta}) = \cos{\alpha}\cos{\beta} - \sin{\alpha}\sin{\beta}$
  • $\cos({\alpha - \beta}) = \cos{\alpha}\cos{\beta} + \sin{\alpha}\sin{\beta}$
  • $\sin({\alpha + \beta}) = \sin{\alpha}\cos{\beta} + \cos{\alpha}\sin{\beta}$
    유도
    • $\cos({\alpha - \beta})$ 공식에서 $\alpha$대신 $\frac{1}{2}\pi - \alpha$를 대입하면,
      • 좌변은 $\cos({\frac{1}{2}\pi - (\alpha + \beta)})$는 $y=x$축 대칭 공식에의해서 $\sin({\alpha + \beta})$이고,
      • 우변은 $\cos({\frac{1}{2}\pi - \alpha})\cos{\beta} + \sin({\frac{1}{2}\pi - \alpha})\sin{\beta} = \sin{\alpha}\cos{\beta} + \cos{\alpha}\sin{\beta}$
  • $\sin({\alpha - \beta}) = \sin{\alpha}\cos{\beta} - \cos{\alpha}\sin{\beta}$

2.1. 덧셈정리 파생

  • 덧셈 정리끼리 더하거나 빼서 정리하면,
    • $\cos{\alpha}\cos{\beta} = \frac{1}{2}(\cos({\alpha + \beta}) + \cos({\alpha - \beta}))$
    • $\sin{\alpha}\sin{\beta} = -\frac{1}{2}(\cos({\alpha + \beta}) - \cos({\alpha - \beta}))$
    • $\sin{\alpha}\cos{\beta} = \frac{1}{2}(\sin({\alpha + \beta}) + \sin({\alpha - \beta}))$
    • $\cos{\alpha}\sin{\beta} = \frac{1}{2}(\sin({\alpha + \beta}) - \sin({\alpha - \beta}))$

  • 위 식에서 $\alpha + \beta = p$, $\alpha - \beta = q$로 놓으면, $\alpha = \frac{p+q}{2}$, $\beta = \frac{p-q}{2}$이므로,
    • $\cos{p} + \cos{q} = 2\cos{\frac{p+q}{2}}\cos{\frac{p-q}{2}}$
    • $\cos{p} - \cos{q} = -2\sin{\frac{p+q}{2}}\sin{\frac{p-q}{2}}$
    • $\sin{p} + \sin{q} = 2\sin{\frac{p+q}{2}}\cos{\frac{p-q}{2}}$
    • $\sin{p} - \sin{q} = 2\sin{\frac{p-q}{2}}\cos{\frac{p+q}{2}}$


3. 배각공식/반각공식

  • 덧셈 공식에서 $\alpha = \beta = \theta$라고 놓으면,
    • $\cos({2\theta}) = \cos^2{\theta} - \sin^2{\theta}$
      • $\cos({2\theta}) = 2\cos^2{\theta} - 1$ $(\because \cos^2{\theta} + \sin^2{\theta} = 1$)
      • $\cos({2\theta}) = 1 - 2\sin^2{\theta}$
    • $\sin({2\theta}) = 2\sin{\theta}\cos{\theta}$

  • 위 식을 정리하면,
    • $\cos^2{\frac{\theta}{2}} = \frac{1 + \cos(\theta)}{2}$
    • $\sin^2{\frac{\theta}{2}} = \frac{1 - \cos(\theta)}{2}$


4. 합성공식

  • 요점: 코사인 함수와 사인함수를 선형 결합하면, 다양한 주파수/파형을 가진 가진 모든 파동 함수들을 만들 수 있다!
  • $a\cos{x} + b\sin{x} = \sqrt{a^2 + b^2}\cos({x -\theta})$
    • 단, $\cos{\theta} = \frac{a}{\sqrt{a^2 + b^2}}$, $\sin{\theta} = \frac{b}{\sqrt{a^2 + b^2}}$
  • $a\cos{x} + b\sin{x} = \sqrt{a^2 + b^2}\sin({x + \theta})$
    • 단, $\cos{\theta} = \frac{b}{\sqrt{a^2 + b^2}}$, $\sin{\theta} = \frac{a}{\sqrt{a^2 + b^2}}$
