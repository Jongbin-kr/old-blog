[미적분009] 다양한 미분공식들

Jongbin Won  included in Mathematics
 2023-07-26  237 words   One minute 
Contents

  • 2. 지수함수의 미분법
  • 3. 로그함수의 미분법
  • 4. 삼각함수의 미분법

1. 일반적인 미분공식들

  • 더하기/뺴기
    • $f(x) \pm g(x) = f^{\prime}(x) \pm g^{\prime}(x)$

  • 상수배
    • $(cf(x))^{\prime} = 0 + cf^{\prime}(x) = cf^{\prime}(x)$

  • 곱하기(곱의 미분법)
    • $f(x) \times g(x) = f^{\prime}(x)g(x) + f(x)g^{\prime}(x)$

  • 나누기(몫의 미분법)
    • $\frac{f(x)}{g(x)} = \frac{f^{\prime}(x)g(x) - f(x)g^{\prime}(x)}{g(x)^2}$ (단, $g(x) \neq 0$)
    • $\frac{1}{g(x)} = \frac{-g(x)}{g(x)^2}$

  • 합성함수의 미분(연쇄법칙/chain rule)
    • $(f \circ g)^{\prime}(x) = f^{\prime}(g(x)) \cdot g^{\prime}(x)$


2. 지수함수의 미분법

  • $f(x) = e^x$일 때, $f^\prime(x) = e^x$
  • $f(x) = a^x = (e^{\log a)x}$이므로, 연쇄법칙에 의해 $f^\prime(x) = \log a \cdot (e^{\log a)x} = \log a \cdot a^x$


3. 로그함수의 미분법

  • $f(x) = \log(x)$일 때, $f^\prime(x) = \frac{1}{x}$
  • 만약 밑이 $e$가 아니라면, $f(x) = \log_a x$일 때, $f^\prime(x) = \frac{1}{\log(a) x}$


4. 삼각함수의 미분법

  • $(\sin\theta)^\prime = + \cos\theta$
  • $(\cos\theta)^\prime = - \sin\theta$
  • $(\tan\theta)^\prime = + \sec^2\theta$

  • $(\csc\theta)^\prime = - \csc\theta\cot\theta$
  • $(\sec\theta)^\prime = + \sec\theta\tan\theta$
  • $(\cot\theta)^\prime = - \csc^2\theta$
